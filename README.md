# AmpUp iOS SDK (Binary Distribution)

Public Swift Package distribution repo for AmpUp iOS SDK.

## How it works

- This repo contains only `Package.swift` with a SwiftPM `binaryTarget`.
- Release assets host `AmpUpSDK.xcframework.zip` for each version tag.
- Source code lives in the private repo: `https://github.com/AmpUp-me/ampup-sdk-ios`.

## Add package in Xcode

Use this repository URL in `File -> Add Package Dependencies`.
